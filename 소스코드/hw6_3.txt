[hw6_3] 이중연결 리스트
----------------------------------------------------------
hw6_2과 동일한 기능을 하는 프로그램을 이중연결리스트를 이용하여 작성하시오.

즉, main의 정수연결리스트를 를 이중정수연결리스트 로 변경하여 작성하면 됨

*** 단,두가지를 출력하도록 할 것.
(1) 리스트 내용 출력
(2) 리스트 내용 역순 출력
즉, 리스트가 3, 5, 7, 9라면 내용 출력은
3 5 7 9
9 7 5 3

출력을 역순으로 출력할 때 리스트 자체가 변경되어서는 안됨(역순 변경하면 안됨)
단지, 출력만 순서대로/역순으로 두번 해야 함
이는 이중연결리스트가 양방향으로 잘 연결되었는지를 확인하기 위한 것임

----------------------------------------------------------
목적
- 이중 연결 리스트를 이해한다.

----------------------------------------------------------
유의사항
- 적절한 comment
//***************************
// 파일명: ... .java
// 작성자: ...
// 작성일: ...
// 설명:...
//***************************

- 들여쓰기
- 식별자 이름을 규칙에 맞게 적절히 붙일 것
- command-line에서 java -jar 과제코드.jar 로 채점합니다. 철자도 똑같게 이 명령이 제대로 실행되어야 합니다.
- 과제코드와 본인의 이름을 맨 처음 출력

----------------------------------------------------------
제출
- 과제코드.jar 파일
- .java 파일들

----------------------------------------------------------
목적
- 이중 연결 리스트를 이해한다.
